Chicago Police DepartmentSpecial Order S07-01-08
Requests for Confiscated Property
Issue Date:29 April 2014Effective Date:01 May 2014
Rescinds:01 April 1989 Version of S07-05
Index Category:Processing Property
I.Purpose
This directive establishes procedures for requesting confiscated property from the Evidence and Recovered Property Section.
II.General Information
  • A.Property that is in the possession of the Evidence and Recovered Property Section (ERPS) and is abandoned, unclaimed, or seized and released by the court to the Department may be confiscated and assigned to units for official business.
  • B.Property subject to administrative disposal may be eligible for Department use.
III.Requesting Confiscated Property
  • A.Exempt members can request confiscated property for official use. Exempt member will:
    • 1.complete and submit a Request for Confiscated Property form (CPD-34.502) for the item through the appropriate chain of command to the Commander, ERPS, explaining the unit need for the confiscated property, and
    • 2.forward a copy of the request to the commanding officer, Equipment and Supply Section (ESS).
  • B.The Commander, ERPS, will identify whether the request can be filled through available confiscated inventory.
    • 1.If the item requested is available the Commander, ERPS, will approve the Request for Confiscated Property form and forward a copy of the approved request to:
      • a.the requesting unit, and
      • b.the commanding officer, ESS.
    • 2.When notified that ERPS can fill the Request for Confiscated Property, the requesting unit exempt member or designee will:
      • a.contact ERPS to arrange pick up of the item, and
      • b.notify ESS when the item has been received.
        NOTE:
        If ESS has determined that the property type requires a CPD asset tag, the requesting unit exempt member or designee will obtain the CPD asset tag from ESS.
    • 3.If the item requested is not available, ERPS personnel will notify the requesting unit and ESS that the request cannot be filled. The unfilled Request for Confiscated Property form will be added to the Confiscated Property Pending Request file and retained for twelve months. If the requested item becomes available within the twelve month retention period, the request will be filled.
      NOTE:
      All unfilled requests must be re-submitted after twelve months to remain in the Confiscated Property Pending Request file.
  • C.City agencies requesting confiscated property must submit a written request through their agency head to the Chief, Bureau of Administration, Attention: Commander, Evidence and Recovered Property Section.
    • 1.If the item requested is available, the Commander, ERPS, will:
      • a.approve the written request,
      • b.send a copy of the approved request to the requesting agency,
      • c.arrange with the agency for pick up of the property, and
      • d.arrange issuance of the agency asset tag for the property, if required.
    • 2.If the item requested is not available, the ERPS personnel will notify the requesting agency that the request cannot be filled. The unfilled request report will be added to the Confiscated Property Pending Request file and retained for six months. If the requested item becomes available within the six month retention period, the request may be filled.
    • 3.ERPS personnel will notify the requesting agency that all unfilled requests must be resubmitted after six months to remain in the Confiscated Property Pending Request file.
