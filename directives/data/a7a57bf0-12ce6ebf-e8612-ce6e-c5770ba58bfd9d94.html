Chicago Police DepartmentUniform and Property U04-01-11
Approved Uniform Groups
Issue Date:23 December 2004Effective Date:23 December 2004
Rescinds:D04-63
Index Category:Uniform and Personal Equipment
I.Purpose
This directive:
  • A.designates, effective 01 January 2005, fall and winter uniform groups to be worn by sworn personnel.
  • B.authorizes the use of the:
    • 1.baseball-style cap.
    • 2.Tactel three-season jacket.
    • 3.Tactel round-crown cap.
    • 4.Tactel trooper cap.
II.Conflict Resolution
In the event of a conflict, this directive will take precedence over any current Department directive issued previously to it.
III.Authorized Uniform Groups
  • A.Uniformed sworn members will refer to the attachments to this directive for a full description, according to uniform group, of authorized uniform configurations.
  • B.The Class A uniform is the prescribed Department uniform. Uniformed sworn members are always authorized to wear the Class A uniform.
  • C.Unless specifically stated otherwise for a particular event, uniformed sworn members are authorized, at their discretion, to wear the Class B or Class C uniforms instead of the Class A uniform.
  • D.Department members assigned to units authorized to wear specialized uniforms (e.g., Forensic Services Section, Mounted Unit, Marine Unit) are authorized to wear their specific specialized uniform while performing unit-related duties.
  • E.Mixing and matching between uniform groups is prohibited.
  • F.Regardless of seasonal uniform requirements, sworn uniformed members may wear the short-sleeve shirt when:
    • 1.the outside temperature is 70°F or higher.
    • 2.assigned to administrative duties inside a police facility.
IV.General Information
  • A.The baseball-style cap and cargo pants are not authorized to be worn with the Class A and Class B uniforms.
  • B.The turtleneck and dickey are authorized for use with the Class B and C uniforms.
  • C.The necktie will never be worn with the Class C Uniform.
  • D.When wearing the Class C uniform, the long-sleeve shirt can be worn as an outer garment only with the turtleneck or dickey.
  • E.The sweater is approved for both the Class B and Class C uniforms.
  • F.When wearing the sweater with the Class B uniform, the necktie is optional.
  • G.Rainwear is authorized during appropriate weather for all uniform groups.
V.Attachments
This directive contains the following attachments entitled:
  • A.Authorized Uniform Items for Fall Wear;
  • B.Authorized Uniform Items for Winter Wear.
