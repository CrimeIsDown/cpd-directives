Chicago Police DepartmentGeneral Order G07-01
Inventory System for Property Taken Into Custody
Issue Date:10 January 1981Effective Date:11 January 1981
Rescinds:G81-01
Index Category:Processing Property
I.Purpose
This directive continues an inventory system for the recording, processing and disposal of property taken into custody, including Medical Examiner's and Public Administrator's property. Department members will refer to the Special Order titled "Inventory System for Property Taken Into Custody" for the procedures and other information related to property taken into custody.
II.Policy
  • A.All property which is seized, recovered, found, or otherwise taken into custody by Department members will be inventoried immediately. Members inventorying property in districts or units in which the eTrack System is operational will process all inventories using the eTrack System. Members inventorying property in districts or units in which the eTrack System has not yet become operational will continue to use the Property Inventory formset (CPD-34.523) to process all property. As appropriate, either the eTrack System or the Property Inventory formset will be used to record receipt of all property except for:
    • 1.animals - the Animal Inventory (ACC-3230-3) will be used.
    • 2.motor vehicles - the Motor Vehicle Inventory Report (CPD-34.303) will be used by auto pound personnel. Personal property found in the vehicle will be removed and inventoried by the recovering officer using the eTrack System or, if not yet operational, the Property Inventory formset.
    • 3.arrestee's personal property - the Personal Property Receipt (CPD-11.502) will be used to record only property of value which must be returned to the arrestee at the time of release or transfer. If the property will not be returned at that time, it will be inventoried in the eTrack system or, if not yet operational, by using the Property Inventory formset.
    • 4.property of other agencies received at the Forensic Services Section - a Request for Analysis/Receipt for Exhibits (CPD-33.100) will be prepared by unit personnel. No other inventory is necessary.
    • 5.suspect obscene books and magazines (unless seized with or incident to the execution of a search warrant).
  • B.The recovering officer, unless relieved of the responsibility by the investigating officer, will provide for the disposal of all property which he inventories by:
    • 1.obtaining the watch commander's authorization for the return of property to the rightful owner.
    • 2.obtaining a court order at the first court hearing authorizing the return of the property to the owner or agent or by another approved disposal method.
    • 3.authorizing the Evidence and Recovered Property Section (E&RPS) to dispose of the property according to law when the property is no longer needed as evidence.
  • C.The Asset Forfeiture Section (AFS) Organized Crime Division will assume investigative responsibility for the disposal of all "narcotic and related" monies seized under the forfeiture statutes following a declination of prosecution against said monies. The AFS will notify the Evidence and recovered Property Section (E&RPS) via a To-From-Subject report when said monies are to be released to the person from whom they were originally seized. The aforementioned To-From-Subject report thereby exempts the E&RPS from the requirements of either a court order property release order or the presence of the inventorying officer in order to release said monies.
  • D.Members will not solicit or accept for their own use any property taken into custody.
  • E.After 30 days, the release of money for non-narcotic-related seizures not held in original form does not require the completion of a “Form 54” Property Release Order (CPD-34.554) or the permission of the inventorying officer/investigator.
