Chicago Police DepartmentDepartment Notice D13-05
Campaign to Break the Code of Silence Drop Box Initiative - Pilot Program
Issue Date:08 March 2013Effective Date:11 March 2013
Rescinds:
Index Category:Department Notice
I.PURPOSE
This directive:
  • A.introduces the pilot program “Campaign to Break the Code of Silence Drop Box Initiative” (DBI).
  • B.establishes procedures for collecting and distributing information received through the DBI.
  • C.introduces the:
    • 1.Campaign to Break the Code of Silence Drop Box Initiative cards (CPD-21.236).
      NOTE:
      The CPD number will not be imprinted on the Campaign to Break the Code of Silence Drop Box Initiative cards.
    • 2.Campaign to Break the Code of Silence Drop Box Initiative Daily Log sheet (CPD-21.237).
    • 3.Campaign to Break the Code of Silence Drop Box Monthly Report (CPD-21.238).
    • 4.Campaign to Break the Code of Silence Drop Box Initiative Transmittal (CPD-21.239).
II.GENERAL INFORMATION
  • A.The Chicago Alternative Policing Strategy (CAPS) Division will administer the DBI.
  • B.The DBI allows stand alone anonymous paper reporting of crime information that can be routinely accessed by the community. The goal of the initiative is to create a safe, anonymous, effective, and efficient mechanism for reporting crime.
  • C.The DBI will begin at 0001 hours on 11 March 2013 in the conflict zones and violence reduction zones until further notice.
III.GEOGRAPHICAL DISTRIBUTION OF BOXES
  • A.District Commanders will ensure box placement locations are within specified venues within the conflict zones and violence reduction zones.
  • B.District CAPS sergeants will ensure:
    • 1.box placement locations are accessible, safe and secure; and
    • 2.boxes are inspected daily.
    • 3.if a box is tampered with or stolen, ensure that the appropriate case report is completed.
      NOTE:
      Send information, including the RD number, via email to: CAPS.dropboxinitiative@chicagopolice.org .
  • C.The CAPS Division will provide a replacement for the missing and/or stolen box to the district within 24 hours of notification.
IV.CUSTODY AND CONTROL OF PRODUCT INFORMATION
  • A.The district station supervisor will ensure:
    • 1.that all drop box keys are kept in a secure location and distributed on a daily basis to the assigned police personnel responsible for collecting and stocking the Campaign to Break the Code of Silence Drop Box Initiative cards. Missing and/or lost keys will be reported to the CAPS Division within 24 hours of discovery via email to: CAPS.dropboxinitiative@chicagopolice.org.
    • 2.police personnel are assigned daily to go to each drop box location in the district and collect the submitted DBI cards (CPD-21.236) and any other information in each box;
    • 3.Campaign to Break the Code of Silence Drop Box Initiative cards are reviewed for any information warranting an immediate police response. If so, the DSS will:
      • a.ensure the dispatch of appropriate Department personnel for preliminary investigation.
      • b.if not in district of occurrence, notify district of occurrence district station supervisor for action.
      • c.notify a Crime Prevention and Information Center (CPIC) supervisor immediately.
      • d.forward the Campaign to Break the Code of Silence Drop Box Initiative cards containing the information with the remaining cards unless directed otherwise by a CPIC supervisor.
      • e.report any activity or actions taken pursuant to the information or threat to CPIC.
    • 4.The district station supervisor will:
      • a.make copies of Campaign to Break the Code of Silence Drop Box Initiative Daily Log sheet (CPD-21.237) for the Campaign to Break the Code of Silence Drop Box Initiative District log book;
      • b.place Campaign to Break the Code of Silence Drop Box Initiative cards in a Department Envelope;
      • c.place the Department Envelope containing Break the Code of Silence Drop Box Initiative cards and Campaign to Break the Code of Silence Drop Box Initiative Daily Log sheet and place in the Evidence Property Envelope (CPD-34.559);
      • d.mark CPIC, Unit 116 as destination;
      • e.heat seal Evidence Property Envelope and sign name and star across seal;
      • f.complete the Campaign to Break the Code of Silence Drop Box Initiative Transmittal form (CPD-21.239);
        • (1)Attach white and pink copy to the Evidence Property Envelope.
        • (2)Forward yellow copy to the CAPS sergeant.
      • g.place sealed Evidence Property Envelope along with Campaign to Break the Code of Silence Drop Box Initiative Transmittal form in the safe/secure location with inventoried money.
      • h.take appropriate actions to ensure prompt delivery of Campaign to Break the Code of Silence Drop Box Initiative cards and Campaign to Break the Code of Silence Drop Box Initiative Daily Log sheet to CPIC;
      • i.participate in policy work group meetings as necessary.
  • B.The district CAPS sergeants will make all reasonable efforts to ensure that the custody and control of all distribution cards are safeguarded by:
    • 1.verifying that delivery of Campaign to Break the Code of Silence Drop Box Initiative cards and Campaign to Break the Code of Silence Drop Box Initiative Daily Log sheet to CPIC had arrived via department mail within the designated timeframe.
    • 2.immediately notify via email to CPIC and the CAPS Division if it is determined that there was a non-delivery and/or loss of Campaign to Break the Code of Silence Drop Box Initiative cards inventory.
  • C.CPIC will, upon receipt, verify the information on the Campaign to Break the Code of Silence Drop Box Initiative Transmittal form and return pink copy to the unit of origin, Attention: CAPS sergeant.
  • D.Police Document Section couriers will collect the packaged Campaign to Break the Code of Silence Drop Box Initiative Cards and Campaign to Break the Code of Silence Drop Box Initiative Daily Log sheet from units and ensure delivery to CPIC, Unit 116.
  • E.Only those members identified in this directive will process and have access to the Campaign to Break the Code of Silence Drop Box Initiative cards.
V.DATA ANALYSIS
The Office of Crime Control Strategies will:
  • A.be the central repository for the information collected and analyzed.
  • B.ensure that all data submitted by the districts is thoroughly vetted.
  • C.ensure that if information requires an immediate response, CPIC forwards the information to the appropriate bureau for follow-up within 24 hours of receipt/discovery.
  • D.identify, document, and adjust for any trends and/or predictable analysis that may exist and report these trends monthly, via email to: CAPS.dropboxinitiative@chicagopolice.org.
  • E.ensure that all Campaign to Break the Code of Silence Drop Box Initiative cards are collected and the information reported is electronically saved.
VI.WORK SITE MEETINGS, COMMUNICATION AND INFORMATION SHARING
  • A.The CAPS Division will host a monthly work site meeting with all DBI stakeholders.
    • 1.Meetings will be structured as process meetings in order to determine and identify any salient issues that may impact the program’s success.
    • 2.If changes are warranted, they will be documented, reviewed, and approved by stakeholders prior to implementation.
  • B.The CAPS sergeant will provide updates on the initiative at:
    • 1.Roll Call and,
    • 2.their Beat and/or community meetings.
VII.MONTHLY REPORTS
  • A.CPIC will complete the appropriate sections of the Campaign to Break the Code of Silence Drop Box Initiative Monthly Report and then scan and email to the CAPS Division at CAPS.dropboxinitiative@chicagopolice.org.
  • B.The CAPS Division will develop and disseminate a monthly report. Each report will highlight district trends and outcomes. Data collected will include, but will not be limited to:
    • 1.total number of Campaign to Break the Code of Silence Drop Box Initiative cards distributed by area/district.
    • 2.number of Campaign to Break the Code of Silence Drop Box Initiative cards returned.
    • 3.area where Campaign to Break the Code of Silence Drop Box Initiative cards are returned.
    • 4.number of boxes fully operational.
    • 5.locations with the highest use (libraries, schools, churches, and/or staffed parks).
    • 6.types of crimes/incidents reported.
    • 7.number of offenses referred for investigation.
    • 8.number of offenses investigated.
    • 9.arrest outcomes.
    • 10.number of boxes replaced and its location.
    • 11.changes in box location.
    • 12.jurisdictional transfer of Campaign to Break the Code of Silence Drop Box Initiative cards.
